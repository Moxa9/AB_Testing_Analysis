# A/B Testing Analysis for Website Conversion

## Project Overview

This project analyzes an A/B test conducted on an e-commerce website to determine whether a new website design improves customer conversion rates.

The project includes:

- Data cleaning and validation
- Exploratory Data Analysis (EDA)
- Data visualization
- Statistical hypothesis testing
- Business recommendations
- SQL queries
- Power BI dashboard

---

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Statsmodels
- SQL
- Power BI
- Jupyter Notebook
